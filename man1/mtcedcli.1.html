NAME

mtcedcli - command line spreadsheet program  

SYNOPSIS

mtcedcli [OPTION]...  

DESCRIPTION

mtcedcli provides spreadsheet functionality via a Command Line Interface (CLI). It offers exactly the same functions as the GUI based mtCellEdit, but instead uses text commands.  

OPTIONS

--version

Output version information and exit.

-q

Quiet - Don't show 'about' info on startup.

-t

Interpret TAB as text.
