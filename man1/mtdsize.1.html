NAME

mtdsize - directory scan sizes  

SYNOPSIS

mtdsize [ OPTIONS ] [ PATHNAME ]...  

DESCRIPTION

mtdsize prints information about a directory, or list of directories.  

OPTIONS

--version

Output version information and exit.

-s

Output only the summary.
