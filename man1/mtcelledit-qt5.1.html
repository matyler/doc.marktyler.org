NAME

mtcelledit-qt5 - spreadsheet editor  

SYNOPSIS

mtcelledit-qt5 [ OPTIONS ] [ FILENAME ]  

DESCRIPTION

mtcelledit-qt5 is a lightweight spreadsheet editor which is designed to handle simple day to day spreadsheet tasks.  

OPTIONS

--version

Output version information and exit.

-csv

All text files loaded or imported assumed to be CSV.

-prefs STRING

Use this file for program preferences.

-tsv

All text files loaded or imported assumed to be TSV.
 

FILES

~/.cache/libmtcelledit/clipboard.ledger.zip

Used for copy and paste between mtCellEdit instances.

~/.config/mtcelledit-qt5/prefs.txt

Used for storing user preferences.
