NAME

mtcrul-qt6 - Mark Tyler's point Cloud RULer  

SYNOPSIS

mtcrul-qt6 [ OPTIONS ]  

DESCRIPTION

mtcrul-qt6 is a tool for viewing and analysing point cloud data.  

OPTIONS

--version

Output version information and exit.

-db STRING

Use this path to read the database files.

-prefs STRING

Use this file for GUI program preferences (e.g. window position).
 

FILES

~/.config/mtcrul-qt6/prefs.txt

Default file used for storing user preferences.
