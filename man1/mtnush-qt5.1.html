NAME

mtnush-qt5 - big number calculator  

SYNOPSIS

mtnush-qt5 [ OPTIONS ]  

DESCRIPTION

mtnush-qt5 allows you to evaluate infix mathematical expressions using arbitrary precision numbers. Integer, Float, and Rational number modes are supported. Set the preferences to set the precision of Float number mode, and how many digits to show per number in the output text. When saving the number, no digits are snipped.  

OPTIONS

--version

Output version information and exit.
 

FILES

~/.config/mtnush-qt5/prefs.txt

Used for storing user preferences.
