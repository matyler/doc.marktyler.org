NAME

mtsorec - sound recorder  

SYNOPSIS

mtsorec [ OPTIONS ]... [ FILENAME ]  

DESCRIPTION

mtsorec records audio and stores the result in a WAV file.  

OPTIONS

--version

Output version information and exit.

-dev INTEGER

Record from this audio input device. Default is -1 which auto-detects.

-gui

Start the GUI: Space pauses; Escape stops;

-ldev

List audio devices.

-prefs STRING

Use this file for GUI program preferences (e.g. window position).

-v

Verbose.
 

EXAMPLE

mtsorec /tmp/output.wav -gui -dev 2  

FILES

~/.config/mtsorec/prefs.txt

Default file used for storing user preferences.
