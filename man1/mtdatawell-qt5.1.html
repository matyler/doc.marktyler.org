NAME

mtdatawell-qt5 - GUI random data and crypto tool

 

SYNOPSIS

mtdatawell-qt5 [ OPTIONS ] [ SODA FILE ]...

 

DESCRIPTION

mtdatawell-qt5 provides random data and cryptographic functionality via a graphical user interface.

 

OPTIONS

--version

Output version information and exit.

-db STRING

Use this path to store the database files.

-prefs STRING

Use this file for GUI program preferences (e.g. window position).

 

FILES

~/.config/libmtDataWell/

Default path for storing the database files.

~/.config/mtdatawell-qt5/prefs.txt

Default file used for storing user preferences.
