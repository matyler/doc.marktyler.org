NAME

mtnetlog - network activity logger

 

SYNOPSIS

mtnetlog [ OPTIONS ] [ FILENAME ]

 

DESCRIPTION

mtnetlog is a very basic network activity logger for GNU/Linux systems.

When no filename or options are specified, mtnetlog.tsv is used as the log file with a delay of 5 seconds on the ppp0 network.

 

OPTIONS

--version

Output version information and exit.

-delay NUM

Poll the network every NUM seconds.

-network STRING

Log activity for this network.
