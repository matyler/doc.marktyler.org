NAME

mtchls - chunk file information  

SYNOPSIS

mtchls [ OPTIONS ] [ FILENAME ]...  

DESCRIPTION

mtchls prints information about a given chunk file, or list of files.  

OPTIONS

--version

Output version information and exit.
