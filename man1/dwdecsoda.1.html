NAME

dwdecsoda - decode soda file  

SYNOPSIS

dwdecsoda [ [OPTION]... [FILE] ]...  

DESCRIPTION

Decode a soda file and retrieve the original file.  

OPTIONS

--version

Output version information and exit.

-db STRING

Load the database from this directory. Can only be set once before the filenames are passed.

-o STRING

Save output files to this directory. Default = ".".

-v

Verbose.
 

EXAMPLE

dwdecsoda -o soda_out_decoded soda_out/*
