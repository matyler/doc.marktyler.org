NAME

mtraft-qt6 - file analysis  

SYNOPSIS

mtraft-qt6 [ OPTIONS ] [ DIRECTORY ]  

DESCRIPTION

mtraft-qt6 analyses a given directory and counts the files, subdirectories, links/other files and bytes contained in it.

Double click an item or press ENTER to examine this subdirectory in a new tab.  

OPTIONS

--version

Output version information and exit.
 

FILES

~/.config/mtraft-qt6/prefs.txt

Used for storing user preferences.
