NAME

mtgentex - generate text  

SYNOPSIS

mtgentex [ OPTIONS ] FILENAME  

DESCRIPTION

mtgentex prints out text as per the input file. This is useful for generating large sequences for soak testing.  

OPTIONS

--version

Output version information and exit.
 

EXAMPLE

See the source code for an example in ./test/mtcedcli/mtgentex.txt

mtgentex mtgentex.txt > output.txt
