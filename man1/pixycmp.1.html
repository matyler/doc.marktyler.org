NAME

pixycmp - compare 2 images

 

SYNOPSIS

pixycmp [ [OPTION]... [FILE] ]...

 

DESCRIPTION

Compare the 2 input files and see if they are the same type and have the same geometry.

 

OPTIONS

--version

Output version information and exit.

 

EXAMPLE

pixycmp in_a.bmp in_b.png
