NAME

mtpixy-qt5 - image editor

 

SYNOPSIS

mtpixy-qt5 [ OPTIONS ] [ FILENAME ]

 

DESCRIPTION

mtpixy-qt5 is a lightweight pixel image editor which is designed to handle simple day to day tasks such as viewing or editing pixel art PNG files.

 

OPTIONS

--version

Output version information and exit.

-prefs STRING

Use this file for program preferences.

-s

Grab a screenshot on startup.

 

FILES

~/.config/mtpixy-qt5/prefs.txt

Used for storing user preferences.
